Game Rules

How to Play Sudoku

Sudoku is a logic-based number placement puzzle. The objective is to fill a grid with digits so that each column, each row, and each of the sub-grids contains all of the digits from 1 to the grid size (6 for 6x6 grids, 9 for 9x9 grids) exactly once.

Basic Rules

  1. Each row must contain all numbers from 1 to the grid size (6 or 9) without repetition.
  2. Each column must contain all numbers from 1 to the grid size (6 or 9) without repetition.
  3. Each sub-grid (the smaller boxes within the main grid) must contain all numbers from 1 to the grid size (6 or 9) without repetition.
  4. Some cells are pre-filled with numbers - these are clues and cannot be changed.
  5. You must use logic and deduction to determine which numbers go in which cells.

Tips for Solving

  • Start by looking for cells that have only one possible number.
  • Use the process of elimination to narrow down possibilities.
  • Look for patterns in rows, columns, and sub-grids.
  • Take your time and think logically - there's no need to guess.
  • Practice regularly to improve your solving speed and technique.

Grid Sizes

We offer two difficulty levels:

Easy (6x6)

Perfect for beginners. The grid is divided into 6 sub-grids of 2x3 or 3x2 cells.

Hard (9x9)

Classic Sudoku challenge. The grid is divided into 9 sub-grids of 3x3 cells.
